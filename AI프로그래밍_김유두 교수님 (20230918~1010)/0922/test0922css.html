★ embedded 방식으로 색 변경 실습하기! 아래는 텍스트 제어 부분을 실습한 것!

  1. color: 텍스트 색상을 변경한다.
  2. text-align: 텍스트 내용을 수평으로 정렬한다. (left, right, center, justify)
  3. text-decoration: 텍스트에 선을 스타일링하는데 사용된다. (overline, underline, uppercase, lowercase)
  4. text-indent: 문단의 첫 번째 줄을 들여쓰기 하는 데 사용 (마이너스 값도 입력 가능)
  5. font-family: 폰트를 지정하는데 사용
  6. font-style: 텍스트의 스타일을 지정하는데 사용
  7. font-size: 폰트의 크기 지정 (px, pt, % 등의 단위로 사용 가능)
  8. line-height: 행간 높이를 지정
  9. top: 요소를 상단에서 얼마나 떨어트릴지 지정
  10. bottom: 요소를 하단에서 얼마나 떨어트릴지 지정
  11. left: 요소를 좌측에서 얼마나 떨어트릴지 지정
  12. right: 요소를 우측에서 얼마나 떨어트릴지 지정
  13. overflow: 요소 내용이 넘치는 경우 처리 방법을 지정한다. (auto, hidden, scroll)
  14~15. overflow-x / -y: 요소의 가로 및 세로 방향으로 내용이 넘칠 때의 처리 방법을 설정
  16. position: 요소의 위치 지정 방법을 설정한다. (static{기본위치}, relative{현재 위치를 기준으로 이동}, absolute{가장 가까운 상대적 위치 요소를 기준으로 이동}, fixed{뷰포트를 기준으로 고정})
  17. z-index: 요소의 쌓임 순서(z-축 순서)를 지정한다. 값은 양의 정수 또는 auto가 되며, 높은 숫자가 더 상위에 쌓인다.

  2. text-align: 텍스트 내용을 수평으로 정렬한다. (left, right, center, justify)
